## üìã **LEVEL 1: HTML FUNDAMENTALS**

### Core Concepts

1. **HTML Document Structure**

   - `<!DOCTYPE html>` declaration
   - `<html>` element
   - `<head>` vs `<body>` sections
   - Metadata elements (`<title>`, `<meta>`, `<link>`)

2. **Basic Elements & Text Formatting**

   - Headings: `<h1>` to `<h6>`
   - Paragraphs: `<p>`
   - Text formatting: `<strong>`, `<em>`, `<mark>`, `<small>`
   - Line breaks: `<br>` and horizontal rules: `<hr>`
   - Quotations: `<blockquote>`, `<q>`, `<cite>`

3. **Links & Navigation**

   - Anchor tags: `<a>`
   - Relative vs Absolute URLs
   - Internal links (page anchors)
   - Link attributes: `href`, `target`, `rel`

4. **Images & Multimedia**
   - Image tag: `<img>`
   - Attributes: `src`, `alt`, `width`, `height`
   - Picture element: `<picture>`
   - Audio: `<audio>`
   - Video: `<video>`

## üìö **LEVEL 2: CONTENT STRUCTURING**

### Lists

1. **Unordered Lists**: `<ul>`, `<li>`
2. **Ordered Lists**: `<ol>`, `<li>`
3. **Description Lists**: `<dl>`, `<dt>`, `<dd>`

### Tables

1. **Basic Table Structure**
   - `<table>`, `<tr>`, `<td>`, `<th>`
2. **Table Attributes**
   - `colspan`, `rowspan`
   - `<thead>`, `<tbody>`, `<tfoot>`
   - `<caption>`

### Forms

1. **Form Structure**: `<form>`, `action`, `method`
2. **Input Elements**
   - Text inputs: `text`, `email`, `password`, `tel`
   - Number inputs: `number`, `range`
   - Date inputs: `date`, `time`, `datetime-local`
   - Choice inputs: `radio`, `checkbox`, `select`, `datalist`
3. **Other Form Elements**
   - Textarea: `<textarea>`
   - Buttons: `<button>`, `submit`, `reset`
   - Labels: `<label>`, `for` attribute
   - Fieldset & Legend: `<fieldset>`, `<legend>`

## üåê **LEVEL 5: INTEGRATION & BEST PRACTICES**

### Integration with Other Technologies

1. **HTML with CSS**
   - Inline styles: `style` attribute
   - Internal and external CSS
   - Class and ID selectors
2. **HTML with JavaScript**
   - `<script>` tag placement
   - Event attributes: `onclick`, `onload`, etc.
   - Data attributes: `data-*`

### Meta Information

1. **SEO Meta Tags**
   - Description, keywords, author
   - Open Graph tags
   - Twitter Card tags
2. **Viewport & Responsive Meta Tags**
3. **Charset & Language Settings**

### Best Practices

1. **Accessibility (A11y)**
   - ARIA attributes: `role`, `aria-label`, `aria-describedby`
   - Semantic HTML for screen readers
   - Proper heading hierarchy
2. **Performance Optimization**
   - Image optimization attributes
   - Lazy loading: `loading="lazy"`
   - Preloading resources
3. **Code Quality**
   - HTML validation
   - Code formatting and indentation
   - Comments: `<!-- Comment -->`
